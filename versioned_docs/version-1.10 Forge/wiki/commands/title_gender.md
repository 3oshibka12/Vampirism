Change the gender of your lord title.

## Syntax

```
/vampirism title-gender <female>
```

## Arguments

| Parameter | Type      | Description                       | 
|:----------|:----------|:----------------------------------|
| female    | `boolean` | Turn female lord titles on or off |

## Permissions

:::info

Available to everyone

:::
