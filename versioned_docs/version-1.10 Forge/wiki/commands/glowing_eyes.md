Choose if your vampire eyes should be rendered glowing

## Syntax

```
/vampirism glowingEye <on>
```

## Arguments

| Parameter | Type      | Description                |
|:----------|:----------|:---------------------------|
| on        | `boolean` | Turn glowing eye off or on |

## Permissions

:::info

Available to everyone

:::
